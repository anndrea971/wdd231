Site Plan: Maipu SLP Center

Andrea Gonzalez | WDD 231

1. Site Name

Maipu Speech-Language Pathology (SLP) Center

Reason: This name provides a professional, location-based identity that helps local residents in Maipu find specialized clinical services quickly.

2. Site Purpose

To provide a clinical resource hub where patients and parents can find speech therapy services, interact with a "Word of the Day" API for vocabulary building, and access educational milestones.

3. Scenarios

  • Scenario 1: A parent is looking for an articulation specialist for their toddler and needs to see if the clinic offers services for that specific age group.
  • Scenario 2: A teacher wants to find a daily language exercise for their classroom and visits the site to use the "Daily Language Goal" tool.

4. Color Schema

Primary: rgba(11, 42, 74, 1)
Secondary: rgba(10, 194, 255, 1)
Background: #F8F9FA
Color Role Application
Primary Headers, navigation background, and branding.
Secondary Button accents, active nav links, and borders.

5. Typography

Heading Font: Montserrat

Used for all major titles (h1, h2, h3) to provide a modern and trustworthy clinical appearance.

Body Font: Open Sans

Used for all paragraphs and lists. This font ensures high legibility and accessibility for users with communication challenges.
